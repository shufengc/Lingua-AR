# Dependencies (excluded from this portfolio snapshot)

This repository is a portfolio snapshot of the project. It intentionally excludes OpenCVForUnity binaries/models
(e.g., iOS frameworks, static libraries, and DNN assets) due to large file size and potential licensing restrictions.

To run/build the full project:
- Import/install OpenCVForUnity into: Assets/OpenCVForUnity/
- Restore any required DNN assets into: Assets/StreamingAssets/OpenCVForUnity/
